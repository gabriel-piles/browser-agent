# Agent Instructions

Guidelines for coding agents working in this repository.

## Implementation Policy

* Use pydantic as much as possible
* Place pydantic model classes in the domain folder
* Important ONE FILE per class
* Less than 16 lines per method or function
* Less than 11 methods per class
* Important DO NOT create __init__.py files
* Important DO NOT create __all__ variables in any case
* Use constants in the script, never use arguments
* Do not use arguments in scripts, use constants in the script


## Testing Policy

Do not implement tests or run tests if not explicitly asked
